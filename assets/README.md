# Assets Folder

## Profile Photo

Please add your profile photo to this folder with the filename: `profile-photo.jpg`

The image should be:
- Format: JPG or PNG
- Recommended size: 500x500 pixels or larger (square aspect ratio)
- The image will be automatically optimized and displayed in the About Me section

## Current Image

The About Me section is currently set to use: `assets/profile-photo.jpg`

Once you add the image file here, it will automatically appear on the website.
